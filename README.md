# NWJS PiP

Quick and simple NW.js project for opening and manipulating multiple browser instances.
Can easily extend functionality for more advanced features and DOM manipulation.

## Instructions

- Download NW.js from https://nwjs.io/
- Change path in start.bat to `path/to/nw.exe .`, depending on where the files were extracted
- run `./start.bat` or `npm start` if you have npm installed (optional)
